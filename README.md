# Smart Grade Visualizer
A web-based application to manage student records, analyze academic performance, and generate interactive visual insights using data-driven techniques.

## Overview
Smart Grade Visualizer is designed to simplify academic performance tracking by enabling users to store student data, analyze subject-wise marks, and visualize trends through intuitive charts.
The project focuses on transforming raw academic data into meaningful insights that help in better understanding student performance.

## Features
* Add and manage student records
* Store subject-wise marks
* Generate interactive performance charts
* Visualize trends and statistics
* Simple and user-friendly interface
* Fast and lightweight application

## Tech Stack
* Backend: Python (Flask)
* Frontend: HTML, CSS
* Data Processing: Pandas
* Visualization: Matplotlib

## Installation & Setup
Follow these steps to run the project locally:
### 1. Clone the repository
```bash
git clone https://github.com/bhavleenkaur-git/smart-grade-visualizer.git
```
### 2. Navigate to the project directory
```bash
cd smart-grade-visualizer
```
### 3. Install dependencies
```bash
pip install -r requirements.txt
```
### 4. Run the application
```bash
python app.py
```
### 5. Open in browser
```
http://127.0.0.1:5000/
```
